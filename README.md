# ⚙️ Instrucciones para trabajar con Git

Todos los integrantes del grupo deben subir sus avances al repositorio usando **Git**.

---

# 📥 1. Clonar el repositorio (solo la primera vez)

Abrir **Git Bash** o la terminal y ejecutar:

```
git clone URL_DEL_REPOSITORIO
```

Ejemplo:

```
git clone https://github.com/ADSO-3312737/nombre-del-proyecto.git
```

Luego entrar a la carpeta del proyecto:

```
cd nombre-del-proyecto
```

---

# ➕ 2. Agregar archivos al repositorio

Cuando agreguen o modifiquen archivos ejecuten:

```
git add .
```

Este comando prepara todos los archivos para el commit.

---

# 💾 3. Crear un commit

Guardar los cambios con un mensaje descriptivo:

```
git commit -m "Descripción del cambio realizado"
```

Ejemplo:

```
git commit -m "Se agregó el algoritmo de cálculo de inventario"
```

---

# 🚀 4. Subir los cambios al repositorio

Enviar los cambios a GitHub:

```
git push
```

---

# 📊 Buenas prácticas

Cada integrante debe realizar commits con su propio avance.

Ejemplos de commits válidos:

```
Estructura inicial del proyecto
Se agregó el diagrama de flujo
Implementación del algoritmo principal
Corrección de errores en el código
Documentación del proyecto
```
